# Claude Auto-Dev Plugin

Autonomous task management for Claude Code.

## Commands

| Command | Description |
|---------|-------------|
| `/auto` | Work through all tasks automatically |
| `/status` | Show task progress |
| `/brainstorm` | Generate new tasks |
| `/continue` | Complete one task then stop |
| `/archive` | Compact prd.json |
| `/clean` | Remove temp files |
| `/stop` | Safe session close |
| `/reset` | Clear stuck state |

## Files

- `prd.json` - Task list
- `prd-archive-*.json` - Archived tasks
- `progress.txt` - Learnings log
